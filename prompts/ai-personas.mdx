## How to use persona prompts

<Steps>
  <Step title="Find a persona">
    Browse the personas above or explore the full prompts in the [source repository](https://github.com/langgptai/awesome-gemini-prompts/tree/main/prompts).
  </Step>
  <Step title="Copy the prompt file">
    Each persona is a standalone `.md` file in the `prompts/` directory of the repository. Copy the full text of the prompt you want to use.
  </Step>
  <Step title="Set it as a system prompt">
    Paste the persona prompt as a system prompt via the Gemini API `system_instruction` parameter, or as the opening context message in a Gemini conversation.

    ```python
    import google.generativeai as genai

    model = genai.GenerativeModel(
        model_name="gemini-2.5-pro",
        system_instruction="[Paste full persona prompt here]"
    )
    ```
  </Step>
  <Step title="Engage in character">
    Start your conversation. The persona will maintain its character, knowledge domain, and communication style throughout the session.
  </Step>
</Steps>

## Tips for effective persona prompting

<Tip>
  For Chinese personas (all of the above except Google Antigravity), add `使用简体中文回答，符合简体中文的表达习惯` to your first message or system prompt to ensure consistent Simplified Chinese responses.
</Tip>

<Tip>
  Some personas are designed for specific domains. For best results:
  - Use **Xiaohongshu personas** when you have real content strategy questions
  - Use **张小龙** when thinking through product design decisions
  - Use **当年明月** when exploring Chinese history in an engaging narrative format
  - Use **诺奖-拉斯洛** when analyzing network effects, organizational dynamics, or success patterns
</Tip>

<Note>
  All persona prompts in this collection are in Chinese. They are optimized for Chinese-language interactions with Gemini. The Google Antigravity prompts are the exception — those are in English.
</Note>

## Creating your own persona

To create a custom persona prompt, include these elements:

```text
# 角色名称 (Persona Name)

## 自我认知 (Self-Definition)
- Input: what the user provides
- Output: what the persona delivers
- Model: which Gemini model to use
- Author: credit

## 知识网络 (Knowledge Network)
[The persona's domain expertise, frameworks, and mental models]

## 交互法则 (Interaction Rules)
[Communication style, tone, what the persona avoids]

## 开场白 (Opening Statement)
[How the persona introduces itself — skip generic "Hi, I'm X" openings]
```
